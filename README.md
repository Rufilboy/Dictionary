# Dictionary
This is just a simple dictonary built with python.

## Usage
Make sure you've python3 installed!, then install git:
```sh
sudo apt install git
```
Clone the repo:
```sh
git clone https://github.com/Rufilboy/Dictionary.git
```
And run:
```sh
cd Dictionary
```
